# object-detection-bbox-art
 バウンディングボックスの装飾用の関数群です。
 
 今までに作ったものが雑多に入っています。

# Requirement
 
* OpenCV 3.4.2
* Pillow 6.1.0
* Tensorflow 1.14.0(sample.pyを動かす場合のみ)
 
# Installation
 
利用したいPythonプログラムと同階層にboundingbox_artディレクトリをコピーしてください。

# Usage
 
サンプルの実行方法は以下です。
 
```bash
python sample.py
```
